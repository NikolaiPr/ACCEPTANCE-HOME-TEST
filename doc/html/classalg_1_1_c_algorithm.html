Detailed Description

template<typename T>
class alg::CAlgorithm< T >

Type specified algorithm class to separate working with templates. Uses any text file with random content as an input, and checks all the words contained in such text file for letter combinations consisting of more than 3 letters. output will consist of up to 10 most common letter combinations and their difference in percentage displayed as a table and a bar chart.
